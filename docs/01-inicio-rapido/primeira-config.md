# ⚙️ Primeira Configuração

*Settings essenciais para começar direito*

## 🎯 Configuração Mínima (2 min)

Vamos configurar apenas o **essencial** para você começar hoje.

### 1. Idioma (Opcional)
```
Tools → Preferences → Basic:
✅ Interface language: Português Brasil
```
*👆 Restart necessário após mudar*

### 2. Sincronização (Importante!)
```
Tools → Preferences → Syncing:
✅ AnkiWeb ID: seu-email@exemplo.com
✅ Password: sua-senha-ankiweb
```

Teste: `Tools → Sync` → deve aparecer "Sync successful"

### 3. Configurações de Revisão
```
Tools → Preferences → Reviewing:
✅ Show play buttons on cards: ✓
✅ Interrupt current audio when answering: ✓
✅ Show remaining card count: ✓
```

## 🏗️ Criar Seu Primeiro Deck

### Passo a Passo
1. **Delete o deck "Default"**
   - Clique na engrenagem ao lado → Delete

2. **Crie um novo deck**
   - Botão `Create Deck`
   - Nome: `01-Frontend-Basico`
   - ✅ OK

### 💡 Dica de Nomeação
```
✅ BOM: 01-JavaScript-Basico
✅ BOM: 02-CSS-Flexbox  
✅ BOM: 03-React-Hooks

❌ RUIM: javascript
❌ RUIM: Estudos
❌ RUIM: Programação
```

## 🎴 Primeiro Card de Teste

Vamos criar um card simples para testar:

1. **Selecione seu deck** `01-Frontend-Basico`
2. **Clique em `Add`** (ou Ctrl+N)
3. **Preencha:**

```
Front (Pergunta):
Qual propriedade CSS centraliza texto horizontalmente?

Back (Resposta):
text-align: center;

Tags: css, texto, alinhamento
```

4. **Clique em `Add`** 
5. **Feche a janela**

## ✅ Teste seu Sistema

1. **Clique no deck** `01-Frontend-Basico`
2. **Clique em `Study Now`**
3. **Deve aparecer sua pergunta**
4. **Pense na resposta e clique `Show Answer`**
5. **Escolha a dificuldade:**
   - `Again` (vermelho) - Se errou
   - `Hard` (amarelo) - Acertou com dificuldade  
   - `Good` (verde) - Acertou normalmente
   - `Easy` (azul) - Muito fácil

## 📊 Configurações de Deck (Importantes)

Selecione o deck → Engrenagem → `Options`:

```
Daily Limits:
✅ New cards/day: 10
✅ Maximum reviews/day: 200

New Cards:
✅ Learning steps: 1m 10m
✅ Interval modifier: 100%

Reviews:
✅ Easy bonus: 130%
✅ Interval modifier: 100%
```

*👆 Essas são as configurações testadas para iniciantes*

## 🔄 Sincronização Final

1. `Tools → Sync` (ou Ctrl+Y)
2. Aguarde "Sync successful"
3. ✅ **Seus dados estão seguros na nuvem!**

## 🆘 Problemas Comuns

**"Sync failed"**
→ Confira email/senha do AnkiWeb

**"Não apareceu pra revisar"**  
→ Deck deve estar selecionado (azul)

**"Botões muito pequenos"**
→ Tools → Preferences → Appearance → UI size: 150%
